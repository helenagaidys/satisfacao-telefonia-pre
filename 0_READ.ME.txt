	Este trabalho consiste em quatro partes. A primeira, com o nome “1_Atividade_Integradora-final.ipynb”, diz respeito ao tratamento da base de dados na linguagem Python. Já a segunda,  “2_Atividade Integradora.Rmd”, é referente ao código de programação para a modelagem dos dados na linguagem R. A terceira, é o arquivo em html com o código da parte dois já rodado. Por fim, a quarta parte, “4_Atividade Integradora-final-Discussao Modelos,ipynb” é a discussão do modelo e fechamento do trabalho.
